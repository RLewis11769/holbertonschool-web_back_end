###### How to use JavaScript files

1. Make sure nodejs is installed in the system
    - If not, install with:
        - sudo apt-install nodejs -y
2. Run setup script specific to that directory
3. Make sure eslint is working
    - If not, install with:
        - npm i -g eslint
4. To run:
    - npm run dev #main.js
